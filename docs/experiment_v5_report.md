## 4. 关键发现

### 4.1 XGBoost Focal 单模型最优

单模型排名（Val AUC-PR）：
- B (XGBoost Focal): **0.9756**
- A (XGBoost 标准): 0.9635
- C (LightGBM): 0.5545 — 过拟合，不宜单独使用

### 4.2 时间平滑效果

| 平滑窗口 | Val AUC-PR | Test F1 |
|---------|-----------|---------|
| 无 | **0.9923** | **0.9569** |
| 3 | 0.9830 | 0.9356 |
| 7 | 0.9704 | 0.9312 |

### 4.3 Selected 子模型无贡献

加入 Selected 子模型几乎无变化，额外复杂度没有价值。

### 4.4 过拟合分析

- **E15 (smooth3)**: Train F1=0.9474, Test F1=0.9356, Δ=+0.0117 — 最稳定
- **E17 (nosmooth)**: Train F1=0.7837, Test F1=0.9569, Δ=-0.1732 — 测试指标高但训练低（阈值极低）
- **LGB (C)**: Train F1=0.1158, Test F1=0.4375 — 严重过拟合

## 5. 推荐方案

### 主推荐：E15 — XGBoost Focal + 平滑窗口 3

```
模型:        XGBoost Focal (scale_pos_weight=480, depth=5, lr=0.03)
时间平滑:    窗口 3
阈值:        0.0061（Val 上最大化 F1 选择）
Test:        AUC-PR=0.9891, F1=0.9356, FP=4, FN=11
```

**理由**：三集指标高度一致，无过拟合迹象。

### 备选：E17 — XGBoost Focal 无平滑

```

```

适合业务上宁可漏检也不能误报的场景（FP=1）。
